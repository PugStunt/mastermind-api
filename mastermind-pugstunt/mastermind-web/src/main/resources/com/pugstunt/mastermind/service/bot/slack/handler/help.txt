
usage: [new game] [start] [start game]
       [guess <guess>] [my guess <guess>] [I guess <guess>]
       [stop] [quit] [help]

These are common MasterMind commands used in various situations:

start a new game - possible commands
   new game
   start
   start game

send a new guess for the ongoing game
   guess
   my guess
   I guess

quit playing / end current game
   stop
   quit
